# 贪吃蛇游戏

这是一个基于 [chuyangliu/snake](https://github.com/chuyangliu/snake) 项目移植的贪吃蛇游戏。游戏的目标是控制蛇不断吃食物，尽可能地填满地图。

## 功能

- 基础贪吃蛇游戏
- 贪婪算法AI控制
- 完整GUI界面

## 安装要求

Python 3.6+ 并安装Tkinter

```
pip install -r requirements.txt
python run.py
```

## 操作说明

- **W/A/S/D**: 控制蛇的方向
- **空格键**: 暂停/继续游戏
- **R**: 重新开始游戏
- **ESC**: 退出游戏
